## Task 1: BMI Calculator

### Objective
The objective of this project is to create a BMI (Body Mass Index) calculator using Python.

### Steps Performed

4. Displayed the BMI result.

### Tools Used

### Outcome
The program successfully calculates BMI and categorizes the user’s health status.

### Objective
The objective of this project is to build a command line application in Python that fetches and displays weather information for a specified location.

- Connected to a weather API to retrieve real-time weather data.
- Processed the API response to extract important weather details.
- Displayed temperature and weather conditions in the terminal.

### Tools Used
Python, Jupyter Notebook, Weather API
